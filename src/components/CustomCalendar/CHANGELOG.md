## [1.0.0] - 2025-01-05

### ✨ 新增功能

#### 核心功能
- 基于 Ant Design 的自定义日历组件
- 支持事件数据展示（EventData 接口）
- 事件徽章点击交互（onEventClick 回调）
- 北京时间自动同步（Asia/Shanghai 时区）
- 当前日期自动高亮（荧光绿圆圈标识）
- 自定义默认显示日期（defaultValue 属性）
- UFO 装饰图标支持（ufoIconSrc 属性）

#### 事件系统
- 单日期支持多个事件徽章
- 事件徽章自定义颜色（Tailwind CSS 类名）
- 事件点击事件冒泡控制
- 事件唯一标识符（id 字段）

#### 样式系统
- 荧光绿主题色（#bbfd3b）
- 深色日历面板（#333333）
- 固定布局（328px × 338px）
- 完全自定义 CSS 支持
- Tailwind CSS 集成

#### TypeScript 支持
- 完整的类型定义（CustomCalendarProps）
- EventData 接口
- BadgeData 接口
- Dayjs 类型支持

## 版本说明

### 版本号规则

格式：主版本号.次版本号.修订号

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- ✨ **新增 (Added)**：新功能
- 🔄 **变更 (Changed)**：现有功能的变更
- ⚠️ **弃用 (Deprecated)**：即将移除的功能
- 🗑️ **移除 (Removed)**：已移除的功能
- 🐛 **修复 (Fixed)**：Bug 修复
- 🔒 **安全 (Security)**：安全相关的修复
